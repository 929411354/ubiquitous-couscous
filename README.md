# ubiquitous-couscous
基于 iOS 的 BSC 多签钱包应用 | 支持 DApp 浏览器、区块链工具、多链交易功能
## 新增高级功能

### 🪙 自定义代币头像系统
1. **IPFS头像上传**
   - 上传代币头像到去中心化存储网络
   - 使用多重IPFS网关确保可用性
   - 头像URL格式: `ipfs://<CID>`

2. **多渠道头像获取**
   - 自动从Trust Wallet资源库获取
   - 支持CoinGecko API搜索
   - 备份从BscScan获取图标

3. **头像选择器**
   - 相册选择或拍照
   - 裁剪和优化功能
   - 使用默认头像选项

### 🌐 节点与ABI管理系统
1. **多节点自动切换**
   - 内置10+官方BSC节点
   - 连接失败时自动切换
   - 支持主网和测试网

2. **ABI智能管理系统**
   - 从BscScan自动获取合约ABI
   - 支持标准代币接口(ERC20, ERC721等)
   - 本地缓存提高性能

3. **自定义合约支持**
   - 轻松添加任意BSC智能合约
   - 自动识别代币标准
   - 灵活的头像上传系统
